# Smart CV Job Recommender

## Description
This project is a web application that allows users to create and manage their CVs, and then receive personalized job recommendations based on their skills, experience, and interests. Users can also select specific sectors such as Finance, Science, or Technology to refine job suggestions.

## Problem
Many job seekers struggle to find relevant job opportunities that match their qualifications and career goals. Searching manually across multiple platforms is time-consuming and inefficient.

## Solution
This app simplifies the job search process by analyzing a user’s CV and automatically recommending suitable job roles. It also allows users to filter jobs by industry sector, making the search more targeted and efficient.

## Features
- CV creation and editing tool
- CV-based job recommendation system
- Sector-based job filtering (Finance, Science, Tech, etc.)
- User-friendly interface
- Potential integration with job listing APIs

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js / Python
- Database: MongoDB / Firebase
- APIs: Job listing APIs

## How to Run
1. Clone the repository
2. Install dependencies
3. Run the development server
4. Open in browser
